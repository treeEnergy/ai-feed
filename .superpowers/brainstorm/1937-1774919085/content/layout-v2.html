设计要点说明

左侧人物列表

  • 「全部动态」按钮 — 显示所有人的混合时间线
  • 点击某人 — 右侧筛选为该人的动态
  • 显示新内容计数红点
  • 支持搜索和添加新关注对象

每周关注主题轴

  • 顶部横向时间轴，按周分段
  • 每周展示关键话题标签
  • 当前周高亮显示
  • 点击某周可快速跳到该时段的内容

动态卡片

  • 中文翻译为主体，原文折叠在下方
  • 标注来源平台（X / GitHub / arXiv / Facebook）
  • AI 自动生成的主题标签
  • 查看原文 / 复制译文等操作

交互逻辑

  • 默认：全部人物的混合时间线
  • 点人物：只看这个人的动态
  • 点时间轴周：跳转到对应时间段
  • 定时后台拉取新内容
